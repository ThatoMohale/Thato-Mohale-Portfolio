## üöÄ Getting Started

Follow these instructions if you wish to clone and run the portfolio locally on your machine.

### Prerequisites

You need to have **Node.js** installed on your system.

### Installation

1.  **Clone the repository:**
    ```bash
    git clone [YOUR_GITHUB_REPO_URL_HERE]
    cd [repository-name]
    ```

2.  **Install dependencies:**
    ```bash
    npm install  # or yarn install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev  # or yarn dev
    ```

4.  Open [http://localhost:3000](http://localhost:3000) (or the port indicated in your console) with your browser to see the result.
